⚙ Smarter Technology — Package Sorter

Robotic arm dispatch simulator

How to use

  1. Enter the package Width, Height, Length (cm) and Mass (kg) on the left panel.
  2. Click Dispatch Package (or press Enter) to sort it into the correct stack.
  3. View results in the stack cards on the right — each dispatched package is logged.
  4. To adjust sorting rules, update the Benchmark Thresholds below the form (changes are saved automatically).
  5. Click Clear Stacks to reset all counters and start fresh.
